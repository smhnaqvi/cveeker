API Endpoints

POST Create User

POST /api/v1/users

Create a new user with professional information, contact details, and career data.

GET Get Users

GET /api/v1/users?page=1&limit=10

Retrieve all users with pagination support for efficient data handling.

POST Create Resume

POST /api/v1/resumes

Create a new resume for a user with comprehensive professional data structures.

GET User's Resumes

GET /api/v1/users/{id}/resumes

Get all resumes belonging to a specific user with full relationship data.

PUT Toggle Status

PUT /api/v1/resumes/{id}/toggle-status

Toggle the active status of resumes for managing multiple resume versions.

POST Clone Resume

POST /api/v1/resumes/{id}/clone

Create a copy of an existing resume for quick customization and variations.
